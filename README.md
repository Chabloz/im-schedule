# im-schedule

PWA schedule for IM

You can find the application here: https://horaires.chabloz.eu/


### clone (or fork)

```sh
git clone https://github.com/Chabloz/im-schedule.git
```

### Install
```sh
npm ci
```

### Dev
```sh
npm run dev
```

### Build
```sh
npm run build
```
